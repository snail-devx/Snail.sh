# Snail.sh

1. 记录一些我经常用到的shell脚本，按照用途非为`core` `docker` `linux`等目录
2. 作为 ``#!/usr/bin/env bash`` 解析运行脚本
3. 仅作为我的linux操作系统安装、初始化、备份、还原等使用

# 1. common 通用脚本

> 基础通用脚本，一些功能的无业务色彩脚本，无日志输出，docker通用脚本等等

1. `core`  ：bash核心脚本，可以被其他脚本引入使用；如日志输出，选项选择等等无业务色彩脚本
2. `docker` ：docker 容器、镜像、数据卷、网络等管理的通用脚本
3. `linux`：linux 系统通用脚本，如 `apt` 软件管理、`systemctl` 服务管理等

# 2.linux-ops 系统运维

> 包括系统安装后的初始化工作、系统备份工作

# 2. services-ops 服务运维

> 操作系统中的一些服务运维管理，如常用的docker 容器、本机nginx等
